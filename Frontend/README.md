# AI Code Reviewer - Frontend

The frontend for the AI Code Reviewer application, built with React and Vite. It provides an intuitive code editor interface where users can paste their code and receive instant AI-powered feedback.

## Features

- **Code Editor:** Integrated with `react-simple-code-editor` for a lightweight coding experience.
- **Syntax Highlighting:** Uses `prismjs` for real-time syntax highlighting of JavaScript code.
- **Markdown Rendering:** Renders AI feedback clearly using `react-markdown` and `rehype-highlight`.
- **Responsive UI:** Clean, split-pane design for writing code on the left and viewing feedback on the right.

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- React 19
- Vite
- Axios
- PrismJS
- React Markdown

## Deployment
The frontend is pre-configured to communicate with the deployed backend hosted on Render.
